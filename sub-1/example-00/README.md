# example-00
This directory contains a small Node.js example using **TensorFlow.js (tfjs-node)** to train a simple neural network from synthetic data. The purpose is purely educational; the code was written for learning purposes.

## Prerequisites 🛠️

Before running the project, make sure you have installed:

1. **Node.js** (recommended v22+ or any current LTS version).
2. **npm** (comes with Node.js).

> If you are on Linux, install via your package manager or download from https://nodejs.org

## Installation 📦

In the terminal, open the `example-00` folder and run:

```bash
npm install
```

This will install `@tensorflow/tfjs-node` and any other dependencies defined in `package.json`.


## Running the code ▶️

There is an npm script configured to automatically restart when the `index.js` file changes:

```bash
npm start
```

Alternatively, you can run it directly:

```bash
node index.js
```
